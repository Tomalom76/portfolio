
### PROJEKTY KURSU AGENTÓW AI

# W trakcie kursu tworzenia agentów AI w programie n8n wykonałem następujące projekty Agentów AI:

- Agent AI Księgowy - analiza faktur, kosztów, sprzedaży
- Agent AI Pomocnik Nauczyciela - tworzenie sprawdzianów o niskim poziomie trudności (dla szkół podstawowych),
                                  koordynacja z chmurą, porównywanie ocen, statystyki
- Agent AI - Marketingowiec
- Agent AI - Prawnik - pomoc chata w obsłudze klientów
- Agent AI - Korepetytor
- Agent AI - Automatyzacja komunikacji z klientami
- Agent AI - Zarządzanie projektami i procesami
- Agent AI - Asystent planowania podróży.

Oprócz podstawowego oprogramowania n8n sprzężonego z dokerem użyto do niektórych agentów także sprzężenia w dokerze za 
pomocą webhook z takimi aplikacjami jak ngrok, OpenWebUi czy elevenLabs.
